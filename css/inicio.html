HU-CREACIÓN DE CATÁLOGO






Descripción



Como usuario que crea el catalogo de productos.



Quiero visualizar los productos que voy a vender.



Para poder identificar los productos que vendo y poder agregarles un inventario.

Criterios de aceptación

  • ID: Se va a capturar con un lector de barras cada uno de los productos o de forma manual para el caso de los ID generados para productos específicos.
  • PRODUCTO: Espacio libre para la persona encargada de la creacion del catalogo pueda hacer una descripción del producto.
  • PRECIO: Este es el precio actual del producto, que más adelante estará enlazado con la orden de compra.
  • IVA: En este espacio habrá un Combobox con las diferentes tarifas (Exento de IVA, 16%, 19%, 22%) que pueden tener los productos y el usuario podrá escoger una de ellas.
  • FOTO: Un espacio donde el usuario que va a crear los productos pueda subir la foto del mismo. Es importante porque esta información aparece en la tienda virtual o APP.
